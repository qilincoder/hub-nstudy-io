环境说明

开发是Windows 10 专业版，系统版本是19043.928，系统语言是中文简体。

vscode版本是1.76.2，系统语言是中文简体。

node版本是v20.10.0。

操作步骤

  1. 管理员打开 Windows PowerShell
  2. 执行 set-ExecutionPolicy RemoteSigned
  3. 选择 Y
  4. 验证是否可用或重启vscode验证是否可用
