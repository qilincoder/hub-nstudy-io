Markdown 支持使用 LaTeX 语法来编写数学公式。你可以在 Markdown 中使用一对美元符号 $ 将 LaTeX 数学公式包裹起来，从而将其渲染为数学公式。

以下是一些常用的 Markdown 数学公式示例：

  1. 行内公式：使用单个美元符号 $ 包裹公式，例如 E=mc2。
  2. 行间公式：使用一对美元符号包裹公式，例如E=mc2
  3. 上标和下标：使用 ^ 表示上标，使用 _ 表示下标，例如 x2 和 xi。
  4. 分式：使用 \frac{}{} 表示分式，例如 12。
  5. 根号：使用 \sqrt{} 表示根号，例如 2。
  6. 求和、积分：使用 \sum 表示求和，使用 \int 表示积分，例如 ∑i=1ni 和 ∫01x2dx。
  7. 矩阵：使用 \begin{matrix} \end{matrix} 表示矩阵，例如 1234。
  8. 其他函数和符号：使用 LaTeX 中的函数和符号来表示，例如 sin⁡(x)、cos⁡(x)、limx→∞ 等。
  9. 点号 (xn)′ = n⋅xn−1
  10. 分支形式表示 $$
    f(x) = \max(0, x) = \begin{cases}
    x, & \text{if } x \geq 0 \
    0, & \text{if } x < 0
    \end

注意，Markdown 渲染引擎需要支持 LaTeX 渲染才能正确显示数学公式。常用的 Markdown 编辑器和平台通常都支持 LaTeX 数学公式的渲染。
